# Variable Declaration

## Explicit Types
``` 
    let x: DataType = value;
    let Name : string = "Sushant";
```

## Type Inference

``` 
    let x = value;
    let Name  = "Sushant";
```
## Dynamic Typing

``` 
    x = value;
    Name  = "Sushant";
```
## Constants

``` 
    const x = value;
    const Name  = "Sushant";
```
